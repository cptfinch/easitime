# Social Return on Investment (SROI) Calculator

## Goal
Quantify the economic value generated by a timebank using SROI methodology.

## Model Inputs
- Number of members
- Average exchanges per member per month
- Types of services exchanged
- Market-equivalent value of services
- Avoided costs (reduced GP visits, reduced social care, reduced loneliness interventions)

## Example Calculation
If 100 members each exchange 2 hours/month:
- 200 hours/month of community service
- At minimum wage (£11.44/hr): £2,288/month market equivalent
- At average service value (~£20/hr): £4,000/month
- Plus avoided NHS costs from reduced loneliness: TBD

## Tools
- Python: Pandas, NumPy
- Sensitivity analysis and Monte Carlo for uncertainty
