# EasiTime Mobile App

## Overview
Modern, mobile-first community currency app. The "Nextdoor for Time Credits" — combining the local community feel of Nextdoor with timebanking, LETS, and optional crypto tokens.

## Key Screens

### 1. Feed / Home
- Local activity feed (recent exchanges, new members, events)
- "Someone near you needs help with..." prompts
- Community announcements

### 2. Discover / Search
- Browse available services by category
- Search for specific skills
- AI-powered suggestions ("Based on your skills, you could help...")
- Map view of nearby requests

### 3. Request / Offer
- Post a new request ("I need help with my garden")
- Post a new offer ("I can teach guitar")
- Set availability, location, estimated time

### 4. Wallet
- TIM balance (time credits)
- LETS balance (mutual credit, if enabled)
- Transaction history
- Fiat bridge (buy credits with cash)
- Cross-community transfer

### 5. Exchange Flow
- Match with a member
- Chat to arrange details
- Confirm exchange (both parties)
- Rate and thank

### 6. Profile
- Bio, skills offered, skills needed
- Exchange history and stats
- DBS badge (if checked)
- Community memberships

### 7. Community
- Community info, members, coordinators
- Events calendar
- Service directory
- Social prescribing referral status (if applicable)

## Design Principles
- **Accessibility first**: Many users will be older adults. Large text, high contrast, simple navigation
- **Offline capable**: Queue actions for sync (rural areas may have poor connectivity)
- **Warm, not corporate**: Community feel, not enterprise software
- **Simple onboarding**: Get started in <2 minutes
